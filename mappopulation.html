Map Population

Interactive tool that calculates the population inside each user-drawn shape. Primary Features include:

  • Sub-Second population aggregation. Through the use of Quadtrees, Cloud-Optimised GeoTIFFs and efficient overview reading through GDAL, the tool can guarantee fast polygon queries (typically < 0.5s).
  • Dynamic-resolution Raster. Dynamically selects dataset resolution & recursion depth based on polygon complexity, to either minimise computational cost or increase accuracy according to use preference.
  • Serverless Backend using AWS. Utilises zip-based AWS Lambda & S3 Rasters to enable scaling without needing the infrastructure of a server.
